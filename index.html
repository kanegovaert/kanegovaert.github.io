My Story

It all started when I got my first Android smartphone for my birthday. I was eleven years old at the time which is way too young for a smartphone. I was an exception compared to the other children. Instead of posting meaningless status updates I wanted to root the phone to run Linux on it. I wanted to customize everything. After a year I got my first laptop. I loved .exe files and wanted to make my own. So I learned C# after trying several different languages. C# was an amazing language for rapidly developing Windows desktop software. After C# I learned some Java, got into website development, and Python. Now I work as a Freelance programmer at the young age of 16. I love programming. Whenever I am not behind my screen smashing my keys I skateboard for fun. I also have a Facebook page, you might be interested in visiting it if you have read this far. Check it out here.
