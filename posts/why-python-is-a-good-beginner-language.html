Python is a very easy general purpose scripting language. Learning how to code can be intimidating. All the languages, all the coding styles, and all the learning resources leaves you miserable. I had this problem too and I am here to say that Python is the best beginner language. That’s because it’s easy so you can immediately see results which motivates you to keep on going.
