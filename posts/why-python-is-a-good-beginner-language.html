Python First

Python is a very easy general purpose scripting language. Learning how to code can be intimidating. All the languages, all the coding styles, and all the learning resources leave you miserable. I had this problem too and I am here to say that Python is the best beginner language. That’s because it’s easy so you can immediately see results which motivates you to keep on going.



But don’t forget

Of course, it isn’t only learning syntax! You should also learn about program structures and what’s happening under the hood. If you don’t know what the difference is between structural programming and object orientated programming you aren’t doing it right.



The quickest way to learn

I know that you want the most blazingly fast way to learn how to code. And in my honest opinion reading a good book is the best way. Look, you can either learn it good the first time or learn small portions and keep on struggling to understand concepts. Just read a book and you’re over with it. But don’t read it while thinking about other stuff. You should be fully concentrated on the materials presented in the book and follow the code examples and see how everything works. I also advice you to take notes for the important stuff and go over it a day later. You should always summarize everything. If you follow the above steps correctly you will learn how to write code.
